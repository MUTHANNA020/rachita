## 🔄 عملية التزامن الكاملة

### 1. رفع البيانات المحلية للسيرفر
```
المسار: BackupAndSyncService.pushLocalDataToServer()
1. جلب جميع البيانات غير المتزامنة locally
2. تحويل الأعداد الصحيحة إلى GUID
3. إضافة الحقول الناقصة (doctorId, lastModifiedAt, إلخ)
4. إرسال POST إلى /Sync/push
5. تحديث حالة is_synced = 1 محلياً
6. حفظ آخر وقت مزامنة
```

### 2. سحب البيانات من السيرفر
```
المسار: BackupAndSyncService.pullServerDataToLocal()
1. إرسال GET إلى /Sync/pull مع lastSyncTime
2. معالجة المرضى: insert or update
3. معالجة الوصفات والأدوية
4. حفظ البيانات محلياً
5. تحديث وقت آخر مزامنة
```

### 3. النسخ الاحتياطي المحلي
```
المسار: BackupAndSyncService.createLocalBackup()
1. جلب جميع البيانات من جميع الجداول
2. حفظ في SharedPreferences مع timestamp
3. الاحتفاظ بآخر backup time للاسترجاع السريع
```
